# Fish-Identification-And-Monitoring-System

Fish Identification and Monitoring System

Overview

The Fish Identification and Monitoring System is designed to improve fish farm management through automated monitoring and detection. Traditional fish monitoring relies on manual observation, which is time-consuming and prone to human error.

This system uses an underwater camera and computer vision techniques to detect and monitor fish in real time. Images captured from the camera are processed using the YOLO object detection algorithm to identify fish and analyze their behavior.

The system communicates with an Arduino Uno using the UART protocol to provide alerts through a buzzer and display information on an LCD screen. This helps fish farmers monitor fish activity efficiently and detect abnormalities at an early stage.

Technologies Used

Python, YOLO, Arduino Uno, Computer Vision, Embedded Systems
